# BabySharkSolder

This project is a keychain baby shark! I've designed it specifically for my baby sister because she absolutely adores the Baby Shark cartoon. Can't wait to attach it to her backpack!

BOM 
- 4x LED's
- 4x Resistors
- 1x Battery cell
- 1x Motor DC
- 1x SW_PUSH
